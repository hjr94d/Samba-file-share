# Samba-file-share
Мини проект: обмен файлами по протоколу samba в linux с использованием клиентского подключения Windows
## ## Цель
Развернуть файловый сервер на Ubuntu Server с доступом через samba, чтобы один пользователь имел полный доступ, а другой не имел.
# Окружение 
- Ubuntu Server (виртуальная машина)
- Windows 10 (клиент)
- Сетевой мост для прямого доступа к виртуалке
## Реализация 
### 1- Созданы пользователи:
 - Misha - доуступ разрешен
 - Misha2 - доуступ запрещен
### 2 - Создана папка /srv/test
   - Группа: filesahare
   - Права: 2770
### 3- Настроеная samba
 - share -
 - доступ разрешен только пользователю misha
 - файлы создаются с маской 0770
### 4- проврека на клиенте
   - под misha можно зайти
   - под misha2 доуступ закрыт
